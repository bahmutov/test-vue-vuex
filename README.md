# Cypress demo test

Testing Vue + Vuex + REST TodoMVC web application using Cypress.

To run tests locally with GUI

```
npm install
npm run open
```

To run tests on CI

```
npm install
npm run ci
```

See CI test artifacts (screenshots, videos) at [Cypress Dashboard](https://dashboard.cypress.io/#/projects/hnzumr/runs)
